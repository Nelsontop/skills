# skills
自用技能合集
| 技能名 | 触发关键字 | 使用场景 | 功能描述 |
|---|---|---|---|
| backend-microservice-development | 后端开发, 微服务, Kotlin, Java, 服务架构 | 设计或开发后端微服务 | 提供后端微服务开发规范，覆盖分层架构、依赖注入、配置管理和 Spring Boot 最佳实践。 |
| bats-testing-patterns | Bats, shell 测试, CI/CD 脚本测试, TDD | 为 shell 脚本建立自动化测试体系 | 提供 Bash Automated Testing System（Bats）测试模式、夹具设计与最佳实践。 |
| coding-agent | 构建新功能, PR review, 大规模重构, 迭代式编码 | 需要委托子代理执行复杂编码任务 | 通过后台进程委托 Codex、Claude Code 或 Pi 代理处理编码任务。 |
| content-creator | 博客, 社媒文案, 营销文案, 标题创作 | 需要内容创作与传播文案 | 生成面向受众的博客、社交媒体和营销内容。 |
| continuous-learning | 会话学习, 模式提取, 自动沉淀, learned skills | 自动从历史会话沉淀可复用经验 | 在会话结束时提取可复用模式并保存为可复用技能。 |
| debugger | 调试, 报错, bug, crash, not working | 定位和修复软件问题 | 提供系统化调试与根因分析流程。 |
| deep-research | 深度研究, 调研, 多来源综述, 引用 | 需要多来源研究与总结 | 进行综合研究并输出带引用的分析结果。 |
| docs-review | 文档评审, markdown diff, 文档 PR | 审阅文档变更质量 | 按 Metabase 文档风格规范审查文档改动。 |
| docs-writer | 写文档, 改文档, docs 目录, .md 文件 | 在仓库内编写或修改文档 | 负责文档写作与编辑，确保结构、风格和准确性。 |
| docx | Word, .docx, 报告, memo, 模板 | 处理 Word 文档输入输出 | 创建、读取、编辑和重构 Word 文档内容与格式。 |
| fact-checker | fact check, verify, 真假核验, 辟谣 | 核查事实真伪 | 基于证据进行事实验证和来源可信度评估。 |
| generate-custom-instructions-from-codebase | 迁移指令生成, 版本差异分析, Copilot 规则 | 大版本迁移或重构时生成一致性改造指令 | 分析两个版本之间的代码演进，生成可用于 Copilot 的迁移与演进指导。 |
| gh-fix-ci | GitHub Actions, PR checks, CI 失败, workflow 日志 | 定位并修复 PR 检查失败 | 使用 `gh` 检查失败任务与日志，提炼修复方案并协助落地。 |
| pdf | PDF, 提取文本, 合并, 拆分, OCR | 处理 PDF 文件任务 | 支持 PDF 读写、拆合并、OCR、表格提取、加密等操作。 |
| security-best-practices | 安全最佳实践, 安全审查, secure by default | 安全编码审查与改进建议 | 按语言和框架给出安全基线检查项与修复建议。 |
| security-ownership-map | 安全归属, bus factor, 敏感代码负责人 | 分析代码安全责任与维护风险 | 基于 git 历史构建人员-文件归属图，识别敏感代码归属风险。 |
| security-threat-model | 威胁建模, 攻击路径, trust boundary | 面向仓库的威胁分析 | 枚举资产、信任边界、攻击路径与缓解措施，输出可执行威胁模型。 |
| sessions | Agent Sessions, 会话窗口, layering, chat widget | 开发 Agent Sessions 窗口相关功能 | 提供 Sessions 架构、目录和开发规范指导。 |
| slide | PPT, 幻灯片, 演示文稿, Markdown 转幻灯片 | 生成演示材料 | 将 Markdown 大纲转换为 HTML 幻灯片演示系统。 |
| spring-boot-engineer | Spring Boot 3.x, WebFlux, Spring Cloud, Security 6 | 构建 Spring Boot 应用与微服务 | 提供 Spring Boot 3.x 与相关生态的工程化支持。 |
| springboot-patterns | REST API, 分层服务, 缓存, 异步, 日志 | Spring Boot 后端架构设计 | 提供 Spring Boot 架构模式与常见后端设计实践。 |
| summarize | 总结, 提炼, transcript, URL 摘要, 视频转写 | 快速摘要文本或音视频内容 | 对 URL、本地文件、播客等内容进行摘要与提取。 |
| technical-writer | 技术文档, README, API 文档, 教程, 使用指南 | 编写面向开发者和用户的技术内容 | 产出清晰的技术文档、接口说明、指南与技术解释内容。 |
| xlsx | xlsx, xlsm, csv, tsv, 表格清洗, 公式 | 电子表格相关任务 | 读取、编辑、清洗、转换和生成电子表格文件。 |
| skill-creator | 创建 skill, 更新 skill, 自定义能力 | 设计新的技能包 | 指导如何创建或更新可复用技能。 |
| skill-installer | 安装 skill, curated skills, GitHub skill | 安装或管理技能 | 将技能从预设列表或 GitHub 仓库安装到本地。 |
